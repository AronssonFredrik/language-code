# Language code
[![npm](https://img.shields.io/npm/v/language-code.svg)](https://www.npmjs.com/package/language-code)

## Getting started

Install the package...

```bash
npm install language-code
```

Use in project:

```JavaScript
import { languageCode } from 'language-code';
...
const language = languageCode['en']
// or
const language = languageCode.en

```
